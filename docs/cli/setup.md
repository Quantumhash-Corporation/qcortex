# `qcortex setup`

Initialize `~/.qcortex/qcortex.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
qcortex setup
qcortex setup --workspace ~/.qcortex/workspace
```

To run the wizard via setup:

```bash
qcortex setup --wizard
```
